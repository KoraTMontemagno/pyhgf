ghgf.pymc.HGFDistribution#

class ghgf.pymc.HGFDistribution(data: numpy.ndarray = array(0.), model_type: str = 'continuous', n_levels: int = 2, response_function: typing.Callable = <function gaussian_surprise>, response_function_parameters: typing.Tuple = ())[source]#

The HGF distribution to be used in the context of a PyMC 4.0 model.

Examples

This class should be used in the context of a PyMC probabilistic model to estimate one or many parameter-s probability density with MCMC sampling.
