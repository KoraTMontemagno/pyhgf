API#

Binary#

Core functionnalities to update binary node structures.

binary_node_update(parameters_structure, ...)

Update the value and volatility parents of a binary node.

binary_input_update(parameters_structure, ...)

Update the input node structure given one observation.

gaussian_density(x, mu, pi)

Gaussian density as defined by mean and precision.

sgm(x[, lower_bound, upper_bound])

Logistic sigmoid function.

binary_surprise(x, muhat)

Surprise at a binary outcome.

Continuous#

Core functionnalities to update continuous node structures.

continuous_node_update(parameters_structure, ...)

Update the value and volatility parents of a continuous node.

continuous_input_update(...)

Update the input node structure.

gaussian_surprise(x, muhat, pihat)

Surprise at an outcome under a Gaussian prediction.

Distribution#

The Herarchical Gaussian Filter as a PyMC distribution. This distribution can be embedded in a PyMC model (pymc>=5.0.0).

hgf_logp(omega_1, omega_2, omega_3, ...[, ...])

Log probability from HGF model(s) given input data and parameter(s).

HGFLogpGradOp([input_data, time, ...])

Gradient Op for the HGF distribution.

HGFDistribution([input_data, time, ...])

The HGF distribution PyMC >= 5.0 compatible.

Model#

The main class used to create a standard Hierarchical Gaussian Filter for binary or continuous inputs, with two or three levels. This class wraps the previous JAX modules and create a standard node structure for these models.

HGF([n_levels, model_type, initial_mu, ...])

The two-level and three-level Hierarchical Gaussian Filters (HGF).

Plots#

Plotting functionnalities to visualize parameters trajectories and correlations after observing new data.

plot_trajectories(hgf[, ci, surprise, ...])

Plot the trajectories of the nodes' sufficient statistics and surprise.

plot_correlations(hgf)

Plot the heatmap correlation of the sufficient statistics trajectories.

plot_network(hgf)

Visualization of node network using GraphViz.

Response#

A collection of responses functions. A response function is simply a callable taking at least the HGF instance as input after observation and returning surprise.

first_level_gaussian_surprise(hgf[, ...])

Gaussian surprise at the first level of a probabilistic network.

total_gaussian_surprise(hgf[, ...])

Sum of the Gaussian surprise across probabilistic network.

first_level_binary_surprise(hgf[, ...])

Sum of the binary surprise along the time series (binary HGF).

Structure#

Utilities for manipulating node structures and applying sequences.

loop_inputs(parameters_structure, data, ...)

Update the node structure after observing one new data point.

apply_sequence(value, time_step, ...)

Apply a predefinded update sequence to a node structure.
